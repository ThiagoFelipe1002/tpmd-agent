# Módulo 8: Combate de Longo Prazo - Perpétuo  
## Aula 11: Iniciante Facebook - Configurando suas Colunas Personalizadas

---

## Resumo

Nesta aula, aprendemos a configurar colunas personalizadas no Gerenciador de Anúncios do Facebook, uma habilidade essencial para gestores de tráfego que desejam analisar campanhas de forma precisa e tomar decisões fundamentadas. O foco foi montar um painel de métricas relevantes para acompanhar o desempenho das campanhas, eliminando dados desnecessários e criando métricas personalizadas, como o **Connect Rate**, que não existe nativamente na plataforma. Também foi mostrado o passo a passo para salvar essa configuração e como criar métricas personalizadas com fórmulas específicas.

---

## 1. Introdução às Colunas Personalizadas no Facebook Ads

- O Gerenciador de Anúncios do Facebook oferece muitas métricas, mas nem todas são relevantes para todos os objetivos.
- Configurar colunas personalizadas permite visualizar apenas as métricas importantes para o seu tipo de campanha.
- Isso facilita a análise, a tomada de decisões e a otimização dos resultados.

---

## 2. Métricas Essenciais para Análise de Campanhas

A configuração recomendada para análise de performance inclui as seguintes métricas, organizadas em uma sequência lógica para facilitar a leitura:

| Métrica                         | Descrição                                                                                  |
|--------------------------------|--------------------------------------------------------------------------------------------|
| Veiculação                     | Status e veiculação da campanha                                                           |
| Ações                         | Ações realizadas pelos usuários (ex: cliques, conversões)                                 |
| Orçamento                     | Valor investido na campanha                                                               |
| Resultados                    | Número total de resultados obtidos                                                        |
| Alcance                       | Número de pessoas únicas que viram o anúncio                                             |
| Impressões                    | Quantidade total de vezes que o anúncio foi exibido                                      |
| Frequência                    | Média de vezes que cada pessoa viu o anúncio                                             |
| CPM (Custo por Mil Impressões) | Custo para mil impressões                                                                |
| Cliques Únicos               | Número de cliques únicos no link do anúncio                                              |
| CPC (Custo por Clique)        | Custo médio por clique no link                                                           |
| CTR (Taxa de Cliques)          | Percentual de pessoas que clicaram no anúncio em relação às impressões                   |
| Visualização de Página de Destino | Quantidade de carregamentos completos da página de destino                              |
| Custo por Visualização de Página | Valor pago por cada visualização da página de destino                                   |
| Initiate Checkout (Finalização Iniciada) | Número de usuários que iniciaram o checkout                                      |
| Custo por Initiate Checkout    | Valor pago por cada início de finalização                                                |
| Compras                       | Número absoluto de conversões de compra                                                  |
| Custo por Compra (CPA)         | Valor pago por cada compra realizada                                                    |
| Valor Usado                   | Total investido no período analisado                                                    |
| ROAS (Retorno sobre o Investimento) | Métrica que indica o retorno financeiro obtido com os anúncios                      |

---

## 3. Passo a Passo para Configuração das Colunas Personalizadas

### 3.1 Acessando o Gerenciador de Anúncios

- Entre na sua conta do Facebook Ads e abra o Gerenciador de Anúncios.

### 3.2 Localizando e Personalizando as Colunas

- Clique na opção **Colunas** no painel principal.
- Selecione **Personalizar Colunas** para montar seu painel do zero.

### 3.3 Seleção e Organização das Métricas

- Adicione as métricas essenciais listadas anteriormente.
- Organize a ordem das métricas arrastando cada item para a posição desejada, por exemplo:
  - Veiculação
  - Ações
  - Orçamento
  - Resultados
  - Alcance
  - Impressões
  - Frequência
  - CPM
  - Cliques Únicos
  - CPC
  - CTR
  - Visualização de Página de Destino
  - Initiate Checkout
  - Custo por Initiate Checkout
  - Compras
  - Custo por Compra
  - Valor Usado
  - ROAS

### 3.4 Salvando a Configuração

- Após finalizar a seleção e organização, clique em **Salvar Colunas** no canto inferior esquerdo.
- Dê um nome identificador para essa configuração personalizada (exemplo: "Colunas Personalizadas Super ADS").
- Confirme o salvamento para que essa configuração fique disponível para uso futuro.

## 4. Criando Métricas Personalizadas no Facebook Ads

### 4.1 Por que criar métricas personalizadas?

- Algumas métricas importantes, como o **Connect Rate**, não existem nativamente no Facebook Ads.
- Métricas personalizadas permitem combinar dados existentes para criar indicadores mais precisos e alinhados com a análise desejada.

### 4.2 O que é o Connect Rate?

- **Connect Rate** é a taxa que mede a qualidade da conexão entre o clique no anúncio e o carregamento efetivo da página de destino.
- Fórmula:  
  \[
  \text{Connect Rate} = \frac{\text{Visualizações de Página de Destino}}{\text{Cliques Únicos no Link}}
  \]
- Expressa em percentual, essa métrica ajuda a identificar problemas como:
  - Lentidão no site
  - Páginas fora do ar
  - Inconsistências entre o anúncio e a página de destino

### 4.3 Passo a passo para criar o Connect Rate

1. No Gerenciador de Anúncios, clique em **Personalizar Colunas**.
2. Vá até a seção **Personalizado**.
3. Clique em **Criar Métrica Personalizada**.
4. Insira a fórmula:  
   `Visualizações de Página de Destino ÷ Cliques Únicos no Link`
5. Defina o formato de exibição como **Percentual**.
6. Configure a visibilidade da métrica (para você ou para toda a equipe).
7. Nomeie a métrica como **Connect Rate**.
8. Clique em **Criar Métrica**.
9. Após criada, busque a métrica digitando "Connect Rate" e adicione-a ao seu painel.

## 5. Exemplos Práticos e Demonstração

- O instrutor demonstrou a configuração das colunas na prática, adicionando e removendo métricas para deixar o painel enxuto e eficiente.
- Mostrou como excluir métricas duplicadas ou irrelevantes para evitar confusão.
- Explicou a importância de incluir métricas como **Orçamento** e **ROAS** para análise financeira.
- Enfatizou que, ao salvar a configuração, o painel estará pronto para receber dados conforme as campanhas forem entregues.
- Destacou que o Connect Rate deve ser criado manualmente, caso não esteja disponível.

---

## 6. Dicas Importantes

- Métricas personalizadas ficam salvas permanentemente na conta do Facebook Ads.
- Você pode criar quantas métricas personalizadas forem necessárias para suas análises específicas.
- Configurar colunas personalizadas transforma o Gerenciador de Anúncios em uma ferramenta poderosa para análise objetiva.
- Um painel bem configurado permite:
  - Identificar rapidamente o desempenho de campanhas, conjuntos e anúncios.
  - Tomar decisões de otimização baseadas em dados concretos.
  - Diagnosticar problemas em diferentes etapas do funil de conversão.
  - Calcular com precisão o retorno sobre o investimento em mídia paga.

---

## 7. Considerações Finais

- A configuração das colunas personalizadas é um passo fundamental para quem deseja dominar o tráfego pago e obter resultados consistentes.
- O painel deve ser simples, mas completo, contendo as métricas que realmente importam para o seu negócio.
- Futuras aulas abordarão configurações mais avançadas e outras estratégias para otimizar ainda mais o acompanhamento das campanhas.

---

## Pontos-Chave

- **Colunas personalizadas** permitem focar nas métricas que importam para sua análise.
- Métricas essenciais incluem: Veiculação, Ações, Orçamento, Resultados, Alcance, Impressões, Frequência, CPM, Cliques Únicos, CPC, CTR, Visualização de Página de Destino, Initiate Checkout, Compras, CPA, Valor Usado e ROAS.
- **Connect Rate** é uma métrica personalizada que mede a efetividade da conexão entre clique e carregamento da página.
- Criar métricas personalizadas é simples e pode ser feito diretamente no Gerenciador de Anúncios.
- Salvar a configuração das colunas facilita o uso contínuo e a análise rápida.
- Um painel bem configurado ajuda a identificar gargalos e otimizar campanhas com base em dados reais.
