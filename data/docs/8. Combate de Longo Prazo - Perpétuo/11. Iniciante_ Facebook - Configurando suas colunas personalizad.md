# Apostila: Configuração de Colunas Personalizadas no Facebook Ads

---

## Resumo

Configurar colunas personalizadas no Gerenciador de Anúncios do Facebook permite visualizar métricas essenciais para análise precisa de campanhas de tráfego pago. A seleção adequada dessas métricas facilita a tomada de decisões estratégicas e a otimização dos resultados. Métricas personalizadas, como o Connect Rate, podem ser criadas para suprir necessidades específicas de análise que não estão disponíveis nativamente na plataforma. O painel personalizado deve conter dados organizados de forma lógica para facilitar a interpretação e o acompanhamento do desempenho das campanhas.

---

## 1. Introdução às Colunas Personalizadas

- Colunas personalizadas são painéis configuráveis no Gerenciador de Anúncios que exibem métricas relevantes para análise de campanhas.
- Permitem eliminar informações desnecessárias e focar nos dados que impactam diretamente a performance.
- Essencial para gestores de tráfego que desejam tomar decisões fundamentadas e otimizar investimentos em mídia paga.

---

## 2. Métricas Essenciais para Análise de Campanhas

### 2.1 Métricas Recomendadas e suas Descrições

| Métrica                        | Descrição                                               |
|-------------------------------|---------------------------------------------------------|
| Veiculação                    | Dados sobre a veiculação dos anúncios                   |
| Ações                         | Ações realizadas pelos usuários (ex: cliques, conversões)|
| Impressão                     | Quantidade de vezes que o anúncio foi exibido           |
| Alcance                       | Número de pessoas únicas que visualizaram o anúncio     |
| Frequência                    | Média de vezes que cada pessoa viu o anúncio             |
| CPM (Custo por Mil Impressões)| Valor pago a cada mil impressões                         |
| Cliques Únicos                | Número de cliques únicos no link do anúncio             |
| CPC (Custo por Clique)        | Valor pago por cada clique no link                       |
| CTR (Taxa de Cliques)         | Percentual de cliques em relação às impressões          |
| Visualização de Página de Destino | Quantidade de carregamentos completos da página de destino |
| Custo por Visualização de Página | Valor pago por cada visualização da página de destino  |
| Connect Rate                 | Taxa de conexão entre cliques e visualizações da página (métrica personalizada) |
| Initiate Checkout (Finalização Iniciada) | Número de usuários que iniciaram o checkout        |
| Custo por Initiate Checkout   | Valor pago por cada início de finalização               |
| Compras                      | Número absoluto de conversões de compra                  |
| Custo por Compra (CPA)       | Valor pago por cada compra realizada                      |
| Valor Usado                  | Total investido no período analisado                      |
| ROAS (Retorno sobre Investimento) | Retorno financeiro obtido em relação ao investimento    |

---

## 3. Passo a Passo para Configuração das Colunas Personalizadas no Facebook Ads

### 3.1 Acesso e Início da Configuração

1. Acessar o **Gerenciador de Anúncios** da conta.
2. No painel principal, localizar a opção **Colunas**.
3. Selecionar **Personalizar Colunas** para iniciar a montagem do painel.

### 3.2 Seleção e Organização das Métricas

- Manter as seguintes métricas essenciais:
  - Veiculação
  - Ações
  - Orçamento
  - Resultados
  - Alcance
  - Impressão
  - Frequência
  - CPM
  - Cliques Únicos (antes do CPC)
  - CPC
  - CTR
  - Visualização de Página de Destino
  - Custo por Visualização de Página de Destino
  - Initiate Checkout (Finalização Iniciada)
  - Custo por Initiate Checkout
  - Compras
  - Custo por Compra (CPA)
  - Valor Usado
  - ROAS

- Remover métricas duplicadas ou que possam causar confusão, como **Connect Rate** (antes de criar a métrica personalizada).
- Organizar a ordem das métricas para facilitar a leitura, por exemplo:
  1. Veiculação
  2. Ações
  3. Orçamento
  4. Resultados
  5. Alcance
  6. Impressão
  7. Frequência
  8. CPM
  9. Cliques Únicos
  10. CPC
  11. CTR
  12. Visualização de Página de Destino
  13. Custo por Visualização de Página de Destino
  14. Initiate Checkout
  15. Custo por Initiate Checkout
  16. Compras
  17. Custo por Compra
  18. Valor Usado
  19. ROAS

### 3.3 Salvando a Configuração

- Após selecionar e organizar as métricas, clicar em **Salvar Colunas** no canto inferior esquerdo.
- Dar um nome identificador para essa configuração personalizada.
- Confirmar o salvamento para que a configuração fique disponível para uso futuro.
- A configuração salva poderá ser acessada rapidamente na lista de predefinições.

## 4. Criando Métricas Personalizadas no Facebook Ads

### 4.1 Necessidade de Métricas Personalizadas

- Algumas métricas importantes não estão disponíveis nativamente, como o **Connect Rate**.
- Métricas personalizadas permitem combinar dados existentes para criar indicadores específicos.

### 4.2 O que é o Connect Rate?

- Métrica que indica a qualidade da conexão entre o clique no anúncio e o carregamento efetivo da página de destino.
- Fórmula:  
  \[
  \text{Connect Rate} = \frac{\text{Visualizações de Página de Destino}}{\text{Cliques Únicos no Link}} \times 100\%
  \]
- Um Connect Rate baixo pode indicar:
  - Problemas de velocidade no site
  - Páginas fora do ar
  - Inconsistências entre a promessa do anúncio e o destino oferecido

### 4.3 Passo a Passo para Criar o Connect Rate

1. Acessar o **Gerenciador de Anúncios** e selecionar **Personalizar Colunas**.
2. Na seção **Personalizado**, clicar em **Criar Métrica Personalizada**.
3. Inserir a fórmula:  
   - Numerador: Visualizações de Página de Destino  
   - Dividido por: Cliques Únicos no Link  
4. Definir o formato de exibição como **Percentual**.
5. Configurar a visibilidade da métrica (para você ou para toda a equipe).
6. Nomear a métrica, por exemplo, **Connect Rate**.
7. Confirmar clicando em **Criar Métrica**.
8. Após criada, buscar a métrica pelo nome e adicioná-la ao painel personalizado.

### 4.4 Dica Importante

- Métricas personalizadas ficam salvas permanentemente na conta.
- É possível criar quantas métricas personalizadas forem necessárias para análises específicas.

## 5. Utilização e Benefícios das Colunas Personalizadas

- Facilita a identificação rápida do desempenho de campanhas, conjuntos e anúncios.
- Permite tomar decisões de otimização baseadas em dados concretos.
- Ajuda a diagnosticar problemas em diferentes etapas do funil de conversão.
- Proporciona cálculo preciso do retorno sobre o investimento em mídia paga.
- À medida que as campanhas são veiculadas, os dados são atualizados automaticamente nas colunas configuradas.

---

## Pontos-Chave

- Configurar colunas personalizadas é fundamental para análise eficiente de campanhas no Facebook Ads.
- Métricas essenciais incluem dados de veiculação, ações, custos, conversões e retorno sobre investimento.
- Organizar as métricas em sequência lógica facilita a interpretação dos dados.
- Métricas personalizadas, como o Connect Rate, ampliam a capacidade analítica da plataforma.
- Salvar configurações personalizadas permite reutilização e agilidade na análise.
- Monitorar o Connect Rate ajuda a identificar gargalos na jornada do usuário entre clique e conversão.
- A análise orientada por dados aumenta a eficiência e a rentabilidade das campanhas de tráfego pago.
