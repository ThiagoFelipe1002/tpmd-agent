# Apostila: Frente de Batalha - Entendendo e Criando o Pixel no Facebook

---

## Resumo

Pixel é um bloco de código instalado no back-end do site para rastrear e reportar comportamentos dos visitantes. Atua como um espião que captura eventos como cliques, rolagem e submissão de formulários, enviando esses dados para a plataforma Meta (Facebook). A inteligência gerada pelo Pixel permite otimizar campanhas de tráfego pago, aumentando vendas e receitas. A criação do Pixel pode ser feita pelo Portfólio Empresarial ou pelo Gerenciador de Eventos, ambos acessíveis via Gerenciador de Anúncios.

---

## 1. Introdução ao Pixel

### 1.1 O que é o Pixel?

- Pixel é um **bloco de código** que deve ser instalado no **back-end** do seu site.
- **Front-end**: parte visível do site, o que o visitante vê.
- **Back-end**: código que faz o site funcionar e onde o Pixel é inserido.
- Apesar de ser um código, sua implementação é simples e compatível com diversas plataformas.

### 1.2 Plataformas compatíveis para instalação do Pixel

- Hotmart Pages
- LeadPages
- João Baus
- Elementor
- WordPress
- HTML customizado

---

## 2. Função e Importância do Pixel

### 2.1 O Pixel como "espião inteligente"

- Monitora e reporta todas as ações realizadas pelos visitantes dentro do site.
- Cada ação capturada é chamada de **evento**.

### 2.2 Eventos rastreados pelo Pixel

- Rolagem da página
- Cliques em elementos
- Submissão de formulários
- Visualização de vídeos e percentual assistido

### 2.3 Fluxo de funcionamento do Pixel

| Passo | Descrição                                  |
|--------|--------------------------------------------|
| 1      | Visitante acessa a página                   |
| 2      | Pixel rastreia ações (eventos)              |
| 3      | Dados são enviados para a plataforma Meta   |
| 4      | Plataforma aprende padrões de comportamento |

### 2.4 Inteligência gerada pelo Pixel

- Com o acúmulo de dados, a plataforma identifica padrões de comportamento:
  - Quem são os compradores
  - Quem se cadastra como lead
  - Como o público interage com o site
- Quanto mais dados qualificados o Pixel coleta, mais inteligente e eficiente ele se torna.
- Pixel inteligente gera mais vendas e receitas.

## 3. Criando o Pixel no Facebook

### 3.1 Considerações iniciais

- O Pixel é fundamental para campanhas de tráfego pago.
- Sem o Pixel, não há dados para otimizar anúncios, o que prejudica resultados.
- Pode-se criar um único Pixel para vários produtos ou nichos, dependendo da estratégia.

### 3.2 Dois caminhos para criar o Pixel

| Caminho                      | Localização                                   | Passos principais                                                                                  |
|-----------------------------|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| **Portfólio Empresarial**    | Menu lateral → Conjunto de Dados              | 1. Acessar Portfólio Empresarial<br>2. Selecionar Conjunto de Dados<br>3. Clicar em Adicionar<br>4. Nomear o Pixel<br>5. Criar |
| **Gerenciador de Eventos**  | Gerenciador de Anúncios → Todas as Ferramentas → Gerenciador de Eventos | 1. Acessar Portfólio Empresarial<br>2. Entrar no Gerenciador de Anúncios<br>3. Selecionar Todas as Ferramentas<br>4. Abrir Gerenciador de Eventos<br>5. Clicar em Conectar Dados<br>6. Selecionar Web<br>7. Nomear o Pixel<br>8. Criar |

### 3.3 Tutorial detalhado: Criando pelo Portfólio Empresarial

1. Acessar o **Portfólio Empresarial** do Facebook.
2. No menu lateral esquerdo, localizar a opção **Conjunto de Dados** (antigo Pixel).
3. Clicar em **Adicionar**.
4. Inserir o nome do Pixel (exemplo: "Pixel Marketing Digital" ou nome do produto/nicho).
5. Clicar em **Criar**.
6. Pixel criado aparecerá na lista.
7. Possibilidade de atribuir pessoas, parceiros e conectar ativos (configurações avançadas).

### 3.4 Tutorial detalhado: Criando pelo Gerenciador de Eventos

1. Acessar o **Portfólio Empresarial**.
2. Entrar no **Gerenciador de Anúncios**.
3. Clicar em **Todas as Ferramentas** (ícone de grade).
4. Selecionar **Gerenciador de Eventos**.
5. Na tela do Gerenciador de Eventos, clicar em **Conectar Dados**.
6. Escolher a opção **Web** e avançar.
7. Inserir o nome do Pixel.
8. Finalizar a criação.

### 3.5 Observações importantes durante a criação

- Podem surgir solicitações de informações adicionais, como área de atuação da empresa ou formas de pagamento.
- Ler atentamente cada solicitação para garantir conformidade.
- A escolha do nome do Pixel deve ser estratégica:
  - Nome do produto específico
  - Nome do nicho (útil para múltiplos produtos na mesma conta)

## 4. Conceitos Técnicos e Termos Importantes

|----------------|--------------------------------------------------------------------------------------------|
| **Pixel**      | Bloco de código instalado no back-end do site para rastrear ações dos visitantes.          |
| **Evento**     | Ação capturada pelo Pixel, como clique, rolagem, envio de formulário, visualização de vídeo.|

## 5. Importância do Pixel para o Tráfego Pago

- O Pixel é o alicerce para campanhas de anúncios online.
- Sem o Pixel, não há coleta de dados comportamentais.
- Sem dados, a plataforma não consegue otimizar anúncios.
- A inteligência do Pixel gera campanhas mais eficientes e maior retorno sobre investimento.
- O Pixel permite segmentar públicos com base em comportamentos reais.
