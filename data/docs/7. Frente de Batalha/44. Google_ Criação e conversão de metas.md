# Apostila: Google Ads - Criação e Conversão de Metas

---

## Resumo

Configurar corretamente ações de conversão no Google Ads é fundamental para otimizar campanhas e alcançar resultados reais, como vendas e geração de leads. A conversão representa uma ação específica do usuário que deve ser rastreada para que o algoritmo direcione anúncios ao público mais qualificado. Diferenciam-se métricas primárias (objetivo final) e secundárias (comportamentos anteriores), que auxiliam no acompanhamento do funil de vendas. A integração com ferramentas como Google Tag Manager e Hotmart permite automatizar o rastreamento e mensurar o desempenho com precisão.

---

## 1. Introdução às Ações de Conversão no Google Ads

### 1.1 O que são ações de conversão?

Ação de conversão é uma atividade específica que o usuário realiza em seu site ou página de destino, considerada valiosa para o negócio. Serve como parâmetro para otimização das campanhas, permitindo medir resultados reais.

### 1.2 Exemplos comuns de ações de conversão

- Preenchimento de formulário (captura de leads)
- Chegada à página de obrigado após cadastro
- Visualização da página de vendas
- Clique no botão do WhatsApp
- Acesso ao checkout (initiate checkout)
- Conclusão de uma compra

---

## 2. Métodos de Rastreamento de Conversões

### 2.1 Tag do Google Ads

- Código inserido diretamente na página ou via Google Tag Manager (GTM).
- Funciona como o pixel do Meta (Facebook), "ouvindo" ações dos visitantes e reportando ao Google Ads.

### 2.2 Importação via GA4 (Google Analytics 4)

- Vinculação entre GA4 e Google Ads.
- Metas criadas no GA4 são importadas para o Google Ads.
- Não detalhado neste material para evitar complexidade inicial.

---

## 3. Importância da Configuração de Conversões

### 3.1 Aprendizado acelerado do algoritmo

Indicar o objetivo real para o Google permite que o algoritmo identifique perfis com maior probabilidade de conversão, entregando anúncios para pessoas com comportamento de compra.

### 3.2 Mensuração de ROI e ROAS

Permite medir o retorno sobre investimento (ROI) e o retorno sobre o investimento em anúncios (ROAS), avaliando custo versus receita gerada.

### 3.3 Liberação de recursos inteligentes

Habilita funcionalidades avançadas de lances automáticos, como:

- CPA Desejado (Custo por Aquisição)
- Maximização de conversões
- ROAS Alvo (Retorno sobre o investimento em anúncios)

## 4. Métricas Primárias e Secundárias

| Tipo de Métrica   | Função                                           | Exemplo                      |
|-------------------|-------------------------------------------------|------------------------------|
| Métrica Primária  | Resultado prioritário para otimização da campanha | Compra, Lead convertido       |
| Métrica Secundária| Comportamento observado que antecede a conversão principal | Visualização de página, Initiate checkout |

### 4.1 Lógica do funil de conversão

1. Usuário clica no anúncio  
2. Visita a página de vendas (métrica secundária)  
3. Acessa o checkout (métrica secundária)  
4. Conclui a compra (métrica primária)  

### 4.2 Dica estratégica

Criar várias conversões, mas escolher apenas uma como principal para otimização. As demais servem como radar para observação, não para direcionamento.

---

## 5. Analogia para Entender a Configuração Correta

Imagine o Google como um cão farejador treinado para seguir um cheiro específico:

- Cheiro de quem apenas visita a página → traz curiosos  
- Cheiro de quem assiste metade do vídeo → traz interessados  
- Cheiro de quem realmente compra → traz compradores reais (objetivo)  

Treinar o algoritmo com o "cheiro" errado resulta em tráfego de baixa qualidade. A conversão correta é o "cheiro" exato do seu melhor cliente.

## 6. Tutorial Prático: Criando Conversões no Google Ads

### 6.1 Criar conversão de compra (métrica primária)

1. Acesse **Ferramentas > Conversões > Nova ação de conversão**  
2. Selecione **Website** e insira o domínio do seu site  
3. Escolha a categoria **Compra**  
4. Defina:  
   - Nome da conversão (ex: Produto-Compra)  
   - Valor (fixo ou variável, ex: R$ 2.500)  
   - Contagem: todas as conversões (para compras múltiplas)  
   - Janela de conversão: 90 dias para cliques  
   - Atribuição: último clique  
   - Conversões otimizadas: ativado  
5. Clique em **Salvar e continuar**  
6. Escolha **Usar o Gerenciador de Tags do Google** para obter:  
   - ID de conversão (ex: AW-XXXXXXXXX)  
   - Rótulo de conversão (label único)  

### 6.2 Configurar integração na Hotmart

1. Acesse **Ferramentas > Pixel de rastreamento**  
2. Selecione o produto desejado  
3. Escolha **Google Ads** e clique em **Adicionar pixel**  
4. Cole o ID de conversão com prefixo "AW-"  
5. Marque **Vendas realizadas**  
6. Cole o rótulo de conversão  
7. Selecione método de pagamento:  
   - Pagamentos imediatos (cartão, débito, saldo Hotmart) — recomendado  
   - Pagamentos não imediatos (PIX, boleto) — menos confiável  
8. Escolha enviar valor da transação  
9. Salve a configuração  

---

### 6.3 Criar conversão de Initiate Checkout (métrica secundária)

1. Crie nova ação de conversão no Google Ads  
2. Selecione categoria **Iniciar finalização de compra**  
3. Nomeie (ex: Produto-Initiate)  
4. Tipo: **Secundária**  
5. Valor: não utilizar  
6. Contagem: todas  
7. Atribuição: último clique  
8. Copie o rótulo de conversão (ID permanece o mesmo)  
9. Na Hotmart, edite pixel existente e adicione novo label para visitas na página de pagamento  
10. Marque **Visitas na página de pagamento**  
11. Salve  

---

### 6.4 Criar conversão de Visualização de Página (métrica secundária)

#### Método 1: Código direto na página

1. Crie ação de conversão no Google Ads  
2. Categoria: **Visualização de página**  
3. Tipo: secundária  
4. Nomeie (ex: Produto-VisuPaginaVendas)  
5. Configure com tag do Google → Ver snippet (código)  
6. Copie o código e cole na página de vendas no carregamento da página  

#### Método 2: Via Google Tag Manager (recomendado)

1. Copie ID de conversão e rótulo do Google Ads  
2. No GTM:  
   - Crie nova tag  
   - Nome: Produto-GA-VisuPaginaVendas  
   - Configuração: Google Ads - Acompanhamento de conversões  
   - Cole ID e rótulo  
3. Crie acionador (trigger):  
   - Tipo: Exibição de página  
   - Condição: Page Path contém [URL única da página de vendas]  
4. Salve tag e acionador  
5. Publique alterações no GTM

## 7. Metas Personalizadas e Colunas Personalizadas

### 7.1 Metas personalizadas

Permitem agrupar múltiplas conversões (primárias e secundárias) em um único conjunto para facilitar a seleção durante a criação de campanhas.

**Como criar:**

1. No Google Ads, acesse **Metas**  
2. Clique em **Adicionar meta personalizada**  
3. Defina nome identificável (ex: Produto-SP)  
4. Selecione as conversões desejadas (ex: Compra, Initiate checkout, Visualização)  
5. Salve  

Na criação da campanha, selecione essa meta personalizada para otimizar pela métrica primária e acompanhar as secundárias.

### 7.2 Colunas personalizadas para acompanhamento

Permitem visualizar o desempenho de cada conversão individualmente no painel de campanhas.

**Como criar:**

1. Na visualização de campanhas, clique em **Colunas > Modificar colunas**  
2. Clique em **Coluna personalizada**  
3. Defina nome (ex: Produto-Initiate)  
4. Selecione métrica base: **Conversões**  
5. Em **Ação de conversão**, busque e selecione a conversão específica  
6. Salve  
7. Arraste a coluna para a posição desejada  
8. Clique em aplicar  

Repita para cada conversão que deseja acompanhar separadamente.

## 8. Checklist para Criação de Conversões no Google Ads

| Passo                                    | Descrição                                                                                  |
|-----------------------------------------|--------------------------------------------------------------------------------------------|
| 1. Acessar menu de conversões            | Ferramentas > Conversões > Nova ação de conversão                                          |
| 2. Selecionar tipo de conversão          | Website, Chamada telefônica ou App                                                         |
| 3. Inserir domínio do site                | Inserir domínio para rastreamento                                                         |
| 4. Escolher categoria da conversão       | Compra, Lead, Visualização de página, etc.                                                 |
| 5. Definir nome da conversão              | Exemplo: Produto-Compra, Produto-Initiate                                                  |
| 6. Definir valor da conversão             | Valor fixo ou variável, conforme o tipo de conversão                                       |
| 7. Configurar janela de conversão         | Padrão: 90 dias para cliques, 1-3 dias para visualização                                   |
| 8. Definir tipo de contagem               | Todas as conversões ou uma por clique                                                     |
| 9. Escolher modelo de atribuição          | Último clique (recomendado para clareza)                                                  |
| 10. Configurar tag                         | Via código manual, Google Tag Manager ou GA4                                              |
| 11. Integrar com plataformas externas     | Exemplo: Hotmart para eventos de compra                                                   |
| 12. Definir métricas primárias e secundárias | Priorizar uma métrica primária para otimização, usar secundárias para análise             |
| 13. Criar metas personalizadas            | Agrupar conversões para facilitar seleção em campanhas                                    |
| 14. Criar colunas personalizadas          | Visualizar métricas específicas no painel de campanhas                                    |

## 9. Conceitos Fundamentais

| Conceito              | Descrição                                                                                     |
|-----------------------|-----------------------------------------------------------------------------------------------|
| Conversão             | Ação específica do usuário que será rastreada e usada para otimização                         |
| Tag do Google Ads     | Código instalado via GTM ou diretamente na página para rastrear conversões                    |
| Métrica Primária      | Objetivo principal da campanha, usado para otimização do algoritmo                            |
| Métrica Secundária    | Comportamento intermediário, usado para observação e análise, não para otimização direta      |
| Meta Personalizada    | Agrupamento de múltiplas conversões para seleção simplificada em campanhas                    |
| Coluna Personalizada  | Visualização individual de cada conversão no painel de campanhas                              |
| Janela de Conversão   | Período em que a conversão será contabilizada após a interação do usuário                     |
| Atribuição            | Modelo que define como o crédito da conversão é distribuído entre os pontos de contato       |
| Acionador (Trigger)   | Condição que determina quando uma tag deve ser disparada no Google Tag Manager               |

---

## 10. Pontos-Chave

- Configurar conversões corretamente direciona o algoritmo para encontrar compradores reais, não apenas curiosos.  
- A conversão é a métrica que o Google Ads usa para otimizar campanhas e medir resultados.  
- Métricas primárias são o foco da otimização; métricas secundárias ajudam a entender o comportamento do funil.  
- Utilizar Google Tag Manager facilita a gestão e implementação das tags de conversão.  
- Integração com plataformas como Hotmart permite rastrear eventos de compra automaticamente.  
- Metas personalizadas agrupam várias conversões para facilitar a gestão e otimização.  
- Colunas personalizadas no Google Ads permitem acompanhar o desempenho individual de cada conversão.  
- Janela de conversão e modelo de atribuição impactam diretamente na análise dos resultados.  
- Revisar e ajustar as configurações de conversão é essencial para manter a eficiência das campanhas.
