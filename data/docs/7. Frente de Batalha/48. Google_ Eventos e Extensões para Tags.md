# Google: Eventos e Extensões para Tags

---

## Resumo

Validação técnica das tags e eventos é essencial para garantir o sucesso das campanhas de tráfego pago. A extensão **Tag Assistant Legacy by Google** permite verificar se as tags do Google Ads, Google Analytics 4 (GA4) e Google Tag Manager (GTM) estão instaladas e disparando corretamente. A verificação prévia evita perdas de leads, vendas e falhas na otimização algorítmica. A instalação e uso da extensão são simples e devem ser parte do checklist antes de qualquer campanha ou lançamento.

---

## 1. Introdução à Validação de Eventos e Tags

- Eventos são comportamentos dos visitantes que devem ser registrados e enviados para plataformas de anúncios.
- Tags, pixels e eventos de conversão precisam estar corretamente instalados e disparando para garantir o rastreamento eficaz.
- A validação técnica assegura que os dados coletados são precisos, evitando perdas e erros que comprometem resultados.

---

## 2. Importância da Validação de Tags e Eventos

### Problemas evitados com validação correta:

| Problema                  | Consequência                                                                 |
|---------------------------|------------------------------------------------------------------------------|
| Perda de leads qualificados | Conversões reais não são registradas, prejudicando a análise de resultados.  |
| Perda de vendas           | Falta de rastreamento impede identificar quais fontes geram receita.         |
| Comprometimento da otimização | Algoritmos recebem dados imprecisos, reduzindo a eficiência das campanhas. |

- A validação é o alicerce invisível para a inteligência das campanhas.
- Deve ser feita antes de qualquer campanha, evento ou lançamento.

---

## 3. Extensão Recomendada: Tag Assistant Legacy by Google

### Função principal

- Verificar se as tags do Google Ads, GA4 e GTM estão instaladas e disparando corretamente.
- Detectar erros, duplicações e problemas de posicionamento das tags.

### Elementos verificados pela extensão

| Elemento                  | O que é verificado                                                        |
|---------------------------|---------------------------------------------------------------------------|
| Tag do Google             | Confirma a presença da tag principal do Google na página.                 |
| Google Tag Manager (GTM)  | Verifica se o container GTM está carregando corretamente.                 |
| Tags de conversão         | Identifica se as tags de conversão são disparadas conforme as ações.      |
| Posicionamento do código  | Avalia se as tags estão inseridas corretamente na `<head>` ou `<body>`.  |

- Caso haja algum problema, a extensão emite um aviso para correção imediata.

---

## 4. Como Instalar a Extensão Tag Assistant Legacy by Google

### Passo a passo

1. Abrir o menu do navegador (ícone de três pontos verticais).
2. Selecionar a opção **Extensões**.
3. Acessar a **Chrome Web Store**.
4. Na barra de pesquisa, digitar: `Tag Assistant by Google`.
5. Selecionar a extensão correta e clicar em **Instalar**.
6. Após a instalação, o ícone da extensão aparecerá na barra de ferramentas do navegador.

---

## 5. Como Utilizar a Extensão

### Procedimento para verificar tags em uma página

1. Acessar a página que deseja validar (exemplo: página de vendas do produto).
2. Clicar no ícone da extensão **Tag Assistant** na barra de ferramentas.
3. Aguardar o carregamento do painel de diagnóstico.
4. Analisar as tags identificadas e seu status.

### Interpretação do painel

- Lista todas as tags Google presentes na página, como:
  - Código do Google Ads (equivalente ao pixel do Facebook).
  - Google Analytics.
  - Google Tag Manager.
- Indica o status de cada tag:
  - Funcionando corretamente.
  - Apresentando erros.
  - Tags duplicadas.
- Permite visualizar os eventos enviados para cada tag clicada.

## 7. Estrutura Típica de Implementação de Tags

- Um container do Google Tag Manager (GTM) — gerenciador central de todas as tags.
- Código do Google Analytics — rastreamento de comportamento e métricas.
- Código do Google Ads (AdWords) — conversões e remarketing.

A extensão permite verificar se essa estrutura está implementada e funcionando corretamente.

---

## 8. Boas Práticas de Validação

- Utilizar a extensão como parte do protocolo padrão antes de qualquer campanha, evento ou lançamento.
- Garantir que:
  - As otimizações algorítmicas recebam dados precisos.
  - Não haja perda de informações cruciais para a tomada de decisão.
- Não é necessário dominar todos os detalhes técnicos da extensão, o essencial é confirmar que as tags estão funcionando e sem erros.
- Dedicar tempo para essa validação técnica é fundamental para o sucesso das campanhas.

---

## 9. Exemplo Prático de Uso

- Ao acessar uma página com a extensão ativa, o painel mostra:
  - Tags do Google Ads instaladas (equivalente ao pixel do Facebook).
  - Presença de múltiplos Google Analytics e GTM (normalmente deve haver apenas um GTM).
- Clicando em cada tag, é possível visualizar os eventos enviados e confirmar o disparo correto.
- Se algum problema ocorrer, a extensão alerta imediatamente para correção.
