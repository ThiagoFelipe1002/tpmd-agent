# Módulo 7 – Frente de Batalha  
## Aula 30 – Facebook: Visualização de Porcentagem de Vídeo no Facebook

---

## Resumo

Esta aula aborda a configuração avançada para rastrear o percentual de visualização de vídeos do YouTube incorporados em páginas específicas, utilizando o Google Tag Manager (GTM) para enviar eventos personalizados ao Meta Ads (Facebook). O objetivo é monitorar o engajamento dos usuários com vídeos em diferentes etapas do funil de vendas, possibilitando a criação de públicos e conversões personalizadas para otimizar campanhas de tráfego pago, remarketing e qualificação de leads.

A metodologia contempla a ativação de variáveis nativas do GTM relacionadas a vídeos, criação de tags HTML personalizadas para disparar eventos de progresso (25%, 50%, 75%, 100%), configuração de acionadores específicos para vídeos do YouTube, delimitação do disparo por URL de página, testes no modo visualização do GTM e a criação de conversões e públicos personalizados na plataforma Meta Ads com filtros baseados no parâmetro dinâmico `video_progress`.

---

## 1. Introdução

### 1.1 Objetivo da Aula

- Explicar a importância de acompanhar a visualização percentual de vídeos em campanhas.
- Demonstrar como configurar o rastreamento de visualização de vídeos no Meta Ads via Google Tag Manager.
- Apresentar a criação de públicos e conversões personalizadas baseadas no percentual de vídeo assistido.
- Mostrar aplicações práticas para otimização de campanhas e qualificação de público.

### 1.2 Contextualização

- O acompanhamento do percentual de vídeo assistido (25%, 50%, 75%, 100%) é uma estratégia refinada para mensurar o engajamento real.
- Permite construir audiências qualificadas para remarketing e otimização de conversões.
- Pode ser aplicado em diferentes cenários: picos de vendas, lançamentos, funis perpétuos e qualificação de públicos.

## 2. Fundamentos e Aplicações Práticas

### 2.1 Objetivo Central do Rastreamento

- Rastrear o comportamento do usuário em relação à visualização de vídeos do YouTube incorporados em páginas específicas.
- Capturar marcos de visualização: 25%, 50%, 75%, 100%.
- Segmentar audiências com base no nível de engajamento demonstrado.
- Aumentar a probabilidade de conversão ao fazer mais pessoas assistirem aos vídeos.

### 2.2 Aplicações Estratégicas

| Cenário                  | Aplicação                                  | Benefício Principal                                   |
|--------------------------|--------------------------------------------|------------------------------------------------------|
| Pico de Vendas e Lançamentos (CPLs/CPPs) | Otimizar campanhas para pessoas que atingem percentual específico | Direcionar anúncios para usuários mais engajados     |
| Funis Perpétuos          | Monitorar nós de conversão até a compra    | Identificar comportamentos qualificados para remarketing |
| Qualificação de Públicos | Criar públicos e conversões por percentual | Segmentar audiências por nível de engajamento         |

### 2.3 Fluxo de Conversão e Nós de Contato

- Exemplo de funil perpétuo:
  1. Clique no anúncio (primeiro contato)
  2. Página de vendas (com vídeo de vendas)
  3. Checkout (decisão de compra)
  4. Compra (conversão final)

- O vídeo na página de vendas é um nó crítico de engajamento.
- Rastrear o percentual assistido permite identificar leads mais qualificados para remarketing.

---

## 3. Premissa Fundamental: O Pixel da Meta

- O pixel da Meta deve estar instalado nas páginas que serão monitoradas.
- Captura todo comportamento registrado na página e envia para a plataforma Meta.
- Implementação:
  - Via Google Tag Manager (centralizador de códigos) com tag HTML personalizada.
  - Ou inserção direta do código do pixel nas páginas.

---

## 4. Configuração no Google Tag Manager (GTM)

### 4.1 Elementos Fundamentais

- **Variáveis**: Capturam dados do vídeo.
- **Tags**: Enviam eventos para o Meta Ads.
- **Acionadores (Triggers)**: Definem quando as tags serão disparadas.

### 4.2 Ativação das Variáveis de Vídeo

No GTM:

1. Acesse o painel **Variáveis**.
2. Clique em **Configurar** (Configuração de variáveis integradas).
3. Na categoria **Vídeos**, ative as seguintes variáveis:
   - `Video Provider`
   - `Video URL`
   - `Video Title`
   - `Video Current Time`
   - `Video Duration`
   - `Video Percent`
   - `Video Status`

Essas variáveis aparecerão na lateral esquerda para uso em tags e acionadores.

### 4.3 Criação da Tag HTML Personalizada

1. No menu principal do GTM, vá para **Tags**.
2. Clique em **Nova Tag**.
3. Nomeie a tag como:  
   `"Evento YouTube Video Progress"`.
4. Selecione o tipo de tag: **HTML Personalizada**.
5. Insira o código de rastreamento no campo HTML, incluindo as tags `<script>` de abertura e fechamento (importante para funcionamento correto).
   
   > **Nota:** O código enviado no mapa mental está incompleto, é necessário acrescentar o `<script>` no início e no final do código.

6. A tag enviará um evento customizado chamado `"YouTube"` para o Facebook, com dados do progresso do vídeo.

### 4.4 Criação do Acionador (Trigger) de Vídeo

1. Ainda na configuração da tag, vá para a seção **Acionadores**.
2. Crie um novo acionador.
3. Nomeie o acionador:  
   `"Progress Vídeo YouTube"`.
4. Selecione o tipo: **Vídeo do YouTube**.
5. Marque as opções:
   - **Iniciar**
   - **Progresso**
   - **Concluído**
6. No campo de progresso, insira os percentuais:  
   `25`, `50`, `75`, `100`.
7. Defina o escopo para **Todos os vídeos** (captura universal).
8. Salve o acionador.

### 4.5 Delimitação por Página Específica

- Para evitar disparo em todos os vídeos do domínio, restrinja o acionador para páginas específicas:
  1. Edite a tag criada.
  2. No acionador, altere de **Todos os vídeos** para **Alguns vídeos**.
  3. Adicione condição:
     - Variável: `Page Path` ou `Page URL`
     - Operador: `contém`
     - Valor: identificador único da página (exemplo: parte da URL, slug)
  4. Salve as alterações.

### 4.6 Teste e Validação no GTM

- Utilize o modo **Visualizar** do GTM para testar:
  1. Clique em **Visualizar**.
  2. Insira a URL da página onde o vídeo está hospedado.
  3. Clique em **Conectar**.
  4. Reproduza o vídeo e observe os eventos de progresso (25%, 50%, 75%, 100%) sendo acionados.
  5. No Tag Assistant, verifique se a tag foi disparada corretamente.
  6. Após validação, publique as alterações no container do GTM.

## 5. Configuração no Meta Ads (Facebook)

### 5.1 Criação de Conversão Personalizada

1. No Gerenciador de Anúncios da Meta, acesse **Gerenciador de Eventos**.
2. Vá para **Conversões Personalizadas**.
3. Clique em **Criar Conversão Personalizada**.
4. Configure:
   - **Nome**: Exemplo `"VTSD - Vídeo 25% - Página de Vendas"`.
   - **Pixel**: selecione o pixel correspondente.
   - **Fonte da ação**: Site.
   - **Evento**: selecione o evento personalizado `"YouTube"` (aparecerá após o envio dos eventos pelo GTM).
5. Adicione filtro:
   - Parâmetro do evento: `video_progress`
   - Operador: `contém`
   - Valor: `"25%"` (ou outro percentual desejado)
6. Salve a conversão personalizada.

- Essa conversão contabilizará automaticamente usuários que atingirem o percentual especificado.
- Pode ser usada para otimizar campanhas com objetivo de conversão baseado em engajamento de vídeo.

### 5.2 Criação de Público Personalizado

1. No Gerenciador de Anúncios, acesse **Públicos**.
2. Clique em **Criar Público Personalizado**.
3. Selecione a fonte: **Site**.
4. Defina o tempo de retenção (máximo 180 dias).
5. Nomeie o público, por exemplo:  
   `"VTSD - Visualização 25% - Página de Vendas"`.
6. Em **Eventos**, selecione o evento `"YouTube"`.
7. Clique em **Refinar por Parâmetro**.
8. Selecione **Dados Personalizados**.
9. Insira o parâmetro: `video_progress`.
10. Configure o operador como `contém` e valor `"25%"`.
11. Crie o público.

- O público será populado automaticamente conforme os usuários atingem o percentual definido.
- Pode ser utilizado para remarketing e criação de públicos semelhantes.

### 5.3 Validação e Monitoramento

- No **Gerenciador de Eventos**:
  1. Acesse **Fontes de Dados** e selecione o pixel.
  2. Localize o evento `"YouTube"`.
  3. Clique para ver detalhes e acesse a **Amostragem de Atividades**.
  4. Observe os eventos individuais e seus parâmetros, especialmente `video_progress`.
  5. Confirme que os valores correspondem aos percentuais configurados (25%, 50%, 75%, 100%).

- Caso os eventos não apareçam:
  - Verifique configurações no GTM (tag, acionador, delimitação de página).
  - Confirme se o pixel está corretamente instalado.

## 6. Síntese do Fluxo Completo

| Etapa | Ferramenta          | Ação                                                                                  |
|--------|---------------------|---------------------------------------------------------------------------------------|
| 1      | Google Tag Manager  | Ativar variáveis de vídeo (Video Provider, URL, Title, Current Time, Duration, Percent, Status) |
| 2      | Google Tag Manager  | Criar tag HTML personalizada com código de rastreamento                              |
| 3      | Google Tag Manager  | Configurar acionador Vídeo YouTube com percentuais 25, 50, 75, 100                   |
| 4      | Google Tag Manager  | Delimitar acionamento por URL ou Page Path para páginas específicas                  |
| 5      | Google Tag Manager  | Testar via modo Visualizar e publicar alterações                                     |
| 6      | Meta Ads            | Criar conversão personalizada filtrada pelo parâmetro `video_progress`               |
| 7      | Meta Ads            | Criar público personalizado com o mesmo filtro                                      |
| 8      | Meta Ads            | Utilizar conversões e públicos para otimização e remarketing                         |

- O parâmetro chave é `video_progress`, que possui valor dinâmico (25%, 50%, 75%, 100%), permitindo múltiplas segmentações a partir de uma única configuração no GTM.
