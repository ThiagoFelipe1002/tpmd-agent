# 7. Frente de Batalha / 4. Facebook - Criando a Conta de Anúncios

---

## Resumo

Criar uma conta de anúncios no Facebook é um passo fundamental para gerenciar campanhas de tráfego pago de forma profissional. O processo inicia com o acesso ao Portfólio Empresarial, que pode ser feito por três caminhos distintos: via Gerenciador de Anúncios, URL direta (business.facebook.com) ou menu de navegação. Após acessar o portfólio, é possível criar uma nova conta de anúncios definindo nome, fuso horário, moeda e finalidade da conta. A conta criada fica vinculada ao portfólio, pronta para uso em campanhas.

---

## 1. Introdução à Conta de Anúncios no Facebook

- A **conta de anúncios** é um dos elementos essenciais para a gestão de tráfego pago dentro da plataforma Meta.
- Representa o ambiente onde são configuradas e gerenciadas as campanhas publicitárias.
- Criar a conta corretamente é fundamental para organizar e profissionalizar a operação de marketing digital.

---

## 2. Acessando o Portfólio Empresarial

O **Portfólio Empresarial** é o ambiente que centraliza os ativos da empresa, incluindo contas de anúncios. Para criar ou gerenciar contas, é necessário acessá-lo. Existem três caminhos principais para isso:

### 2.1 Via Gerenciador de Anúncios

1. Acessar a tela inicial da plataforma Meta.
2. Clicar em **Gerenciador de Anúncios**.
3. O sistema abrirá o gerenciador pessoal vinculado ao perfil.
4. No topo, localizar e clicar no nome do Portfólio Empresarial desejado (exemplo: "Marketing Digital").
5. Clicar no ícone de **Engrenagem** para acessar as configurações do portfólio.

### 2.2 Via URL Direta

1. Abrir uma nova janela no navegador.
2. Digitar a URL: `business.facebook.com`.
3. Aguardar o carregamento da página.
4. Clicar em **Configurações** para acessar o Portfólio Empresarial.

### 2.3 Via Menu de Navegação

1. Na interface do Business Suite ou Gerenciador de Negócios, localizar o menu principal (ícone de três tracinhos).
2. Selecionar a opção **Configurações**.
3. O sistema carregará o ambiente do Portfólio Empresarial.

---

## 3. Criando a Conta de Anúncios

### 3.1 Iniciando o Processo

- Dentro do Portfólio Empresarial, acessar a seção **Contas de Anúncios**.
- Clicar em **Adicionar Conta de Anúncios**.
- A plataforma apresentará três opções:

| Opção                         | Descrição                                                                                             | Quando Utilizar                                      |
|-------------------------------|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------|
| Criar uma nova conta de anúncios | Criar uma conta do zero para gerenciar anúncios da sua empresa, marca ou organização.               | Quando não possuir nenhuma conta de anúncios.       |
| Adicionar uma conta existente  | Vincular ao portfólio uma conta já existente, à qual se possui acesso.                              | Para incorporar contas de terceiros, agências ou clientes. |
| Solicitar acesso a uma conta   | Solicitar permissão para acessar uma conta de anúncios que pertence a outra pessoa ou empresa.      | Quando não se tem acesso direto à conta de anúncios. |

### 3.2 Passo a Passo para Criar uma Nova Conta

1. Selecionar **Criar uma nova conta de anúncios**.
2. Definir o **nome da conta de anúncios** (exemplo: "Conta de Anúncios 01 - MKT"). O nome pode ser alterado posteriormente.
3. Escolher o **fuso horário** adequado à operação (exemplo: São Paulo para Brasil).
4. Selecionar a **moeda** para cobrança e exibição de valores (exemplo: BRL - Real).
5. Clicar em **Avançar**.

### 3.3 Definir a Finalidade da Conta

- Escolher entre:
  - **Para minha empresa**: quando a conta será usada para anunciar produtos ou serviços próprios.
  - **Para outra empresa ou cliente**: quando a conta será usada para gerenciar anúncios de terceiros.
- Selecionar a opção correspondente e clicar em **Avançar**.

### 3.4 Aceitar os Termos e Confirmar

- Ler atentamente os termos de uso apresentados.
- Marcar a opção de concordância.
- Clicar em **Criar Conta de Anúncios**.

### 3.5 Conta Criada

- A conta de anúncios será criada e vinculada automaticamente ao Portfólio Empresarial.
- A conta aparecerá no menu de **Contas de Anúncios** para acesso e gerenciamento futuros.

---

## 4. Configurações Importantes

### 4.1 Fuso Horário e Moeda

- A escolha do fuso horário e da moeda é **definitiva** e **não pode ser alterada** após a criação da conta.
- É fundamental selecionar corretamente para evitar problemas futuros na gestão financeira e de relatórios.

---

## 5. Considerações Finais

- Criar a conta de anúncios é o primeiro passo para iniciar operações de tráfego pago no Facebook.
- O Portfólio Empresarial organiza todos os ativos da empresa, facilitando a gestão profissional e escalável.
- Próximos passos envolvem a organização completa do portfólio e a configuração de campanhas.

---

## 6. Resumo do Processo de Criação da Conta de Anúncios

| Etapa                          | Ação                                                                                  |
|-------------------------------|---------------------------------------------------------------------------------------|
| 1. Acesso                     | Entrar no Portfólio Empresarial via Gerenciador de Anúncios, URL direta ou menu.      |
| 2. Contas de Anúncios         | Navegar até a seção e clicar em "Adicionar Conta de Anúncios".                        |
| 3. Criar nova conta           | Selecionar "Criar uma nova conta de anúncios".                                       |
| 4. Configurar dados básicos   | Definir nome, fuso horário (São Paulo) e moeda (BRL).                                |
| 5. Definir finalidade         | Escolher se é para a própria empresa ou para cliente/terceiros.                      |
| 6. Aceitar termos             | Ler, concordar e confirmar a criação da conta.                                      |
| 7. Conta criada               | Conta vinculada ao portfólio, pronta para uso.                                       |

---

## 7. Pontos-Chave

- Conta de anúncios é o ativo central para gerenciar campanhas no Facebook.
- Portfólio Empresarial é o ambiente que centraliza contas e outros ativos.
- Três formas principais de acessar o Portfólio: Gerenciador de Anúncios, URL direta e menu de navegação.
- Ao criar a conta, nome, fuso horário e moeda devem ser escolhidos com atenção, pois são permanentes.
- A finalidade da conta deve ser definida para uso próprio ou para terceiros.
- A conta criada fica automaticamente vinculada ao Portfólio Empresarial.
- Solicitar acesso ou adicionar conta existente são opções para gerenciar contas de terceiros.
- Organização do portfólio é essencial para uma gestão profissional e escalável do tráfego pago.
