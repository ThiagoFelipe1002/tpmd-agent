## Aula 19: Facebook - Tipos de Públicos

---

## Resumo

Nesta aula, aprendemos detalhadamente sobre os tipos de públicos disponíveis na plataforma Meta (Facebook e Instagram) para campanhas de tráfego pago. Foram explorados os conceitos, funcionalidades e aplicações práticas dos três principais tipos de públicos: **Público Personalizado**, **Público Semelhante (Lookalike)** e **Público Salvo**. A aula abordou como criar, gerenciar e utilizar esses públicos para otimizar campanhas, destacando a importância da retenção, segmentação por comportamento, e estratégias para alcançar audiências qualificadas. Também foram apresentadas as fontes para criação de públicos personalizados, como visitantes de site, listas de clientes, interações em vídeos, Instagram, Facebook, entre outros.

## Introdução

Compreender profundamente o público-alvo é fundamental para o sucesso em estratégias de tráfego pago. Nesta aula, o foco foi mostrar como a plataforma Meta oferece ferramentas para organizar, segmentar e encontrar as melhores oportunidades de audiência para suas campanhas. Através do Gerenciador de Anúncios, é possível criar e gerenciar públicos que potencializam o alcance e a conversão dos anúncios.

---

## Acesso e Navegação na Plataforma de Públicos do Facebook

- No **Gerenciador de Anúncios**, na lateral esquerda, há um menu chamado **Públicos** (ícone com três pontinhos → Todas as ferramentas → Públicos).
- Ao acessar, abre-se uma janela com a lista de públicos já criados.
- É possível filtrar públicos por:
  - **Status** (ativos, inativos, etc.)
  - **Tipo de público** (personalizado, semelhante, salvo)
  - **Disponibilidade**
  - **Fonte de origem**
- Botão principal para criação: **Criar Público** → opções de públicos personalizados e semelhantes.

## Tipos de Públicos na Plataforma Meta

### Público Personalizado

É a ferramenta mais poderosa para estratégias de **remarketing** e **nutrição de leads**, permitindo criar segmentações baseadas em comportamentos específicos dos usuários que já interagiram com seus ativos digitais.

- **Retenção do público:** período máximo de 180 dias para sites (365 dias para vídeos), durante o qual o usuário fica "armazenado" para ser impactado por anúncios.
- Permite criar públicos qualificados para anúncios futuros.

#### Fontes para criação de Público Personalizado

| Fonte                  | Descrição                                              | Uso no Mercado Digital          |
|------------------------|--------------------------------------------------------|--------------------------------|
| Site                   | Visitantes de páginas web com Pixel instalado          | Muito utilizado                 |
| Lista de Clientes      | Base de dados importada (e-mail, telefone, etc.)       | Muito utilizado                 |
| Vídeo                  | Espectadores de vídeos na plataforma                    | Muito utilizado                 |
| Conta do Instagram     | Pessoas que interagiram com o perfil                    | Muito utilizado                 |
| Página do Facebook     | Pessoas que interagiram com a página                     | Utilizado                      |
| Formulário de Lead     | Pessoas que interagiram com formulários                  | Pouco utilizado                 |
| Atividade em Apps      | Usuários de aplicativos móveis                           | Não aplicável para infoprodutos |
| Catálogo               | Interações com catálogo de produtos                      | Específico para e-commerce      |
| Atividades Offline     | Conversões no mundo físico                               | Não aplicável para infoprodutos |

### Público Personalizado por Site

- Utiliza o **Pixel do Facebook** instalado no site para capturar visitantes.
- Pode segmentar:
  - Todos os visitantes do site.
  - Visitantes de páginas específicas (usando fragmentos da URL, ex: "URL contém").
  - Visitantes que realizaram eventos específicos (ex: tempo gasto, cliques, compras).
- **Retenção:** máximo de 180 dias.
- **Exemplo prático:** Criar público para visitantes da página de vendas usando URL parcial para identificar a página.

**Passo a passo para criar:**

1. Acesse a seção de Públicos no Gerenciador de Anúncios.
2. Clique em criar público personalizado.
3. Escolha a fonte "Site".
4. Selecione o Pixel correspondente.
5. Defina o evento a ser rastreado (todos os visitantes ou evento específico).
6. Configure o período de retenção (1 a 180 dias).
7. Nomeie o público (ex: "VTSD - Página de Vendas - 180 dias").
8. Finalize a criação.

---

### Público Personalizado por Lista de Clientes

- Permite importar uma base de dados (e-mails, telefones, nomes) para o Facebook.
- O Facebook realiza um **match** entre os dados importados e os usuários cadastrados na plataforma, usando identificadores únicos:
  - E-mail
  - Telefone
  - Nome e outros dados complementares
- Público ideal para campanhas de **upsell**, **cross-sell** e remarketing para clientes existentes.
- Pode-se baixar um modelo de planilha para preenchimento e importação.
- Exemplo prático: exportar lista de compradores da Hotmart e importar para o Facebook.

**Passo a passo para importar:**

1. Criar público personalizado → Lista de Clientes.
2. Baixar o modelo de arquivo.
3. Preencher com dados disponíveis.
4. Importar o arquivo.
5. Aguardar processamento e correspondência.
6. Nomear e salvar o público.

---

### Público Personalizado por Vídeo

- Segmenta pessoas que assistiram a vídeos publicados na plataforma.
- Permite criar públicos baseados no nível de engajamento, medido pelo percentual de vídeo assistido:

| Percentual de Visualização | Interpretação             |
|----------------------------|--------------------------|
| 3%                         | Visualização inicial (baixo engajamento) |
| 10 segundos                | Retenção mínima          |
| 25%                        | Interesse moderado       |
| 50%                        | Engajamento significativo|
| 75%                        | Alto interesse           |
| 95%                        | Engajamento máximo       |

- Retenção máxima para vídeos: **365 dias**.
- Pode-se selecionar múltiplos vídeos para criar um público combinado.

**Passo a passo para criar:**

1. Criar público personalizado → Fonte "Vídeo".
2. Selecionar percentual de visualização.
3. Selecionar vídeos (um ou vários).
4. Definir retenção (até 365 dias).
5. Nomear o público (ex: "VTSD - Combo Vídeos - 25%").
6. Criar público.

---

### Público Personalizado por Conta do Instagram

- Cria públicos baseados em interações com o perfil do Instagram.
- Tipos de interação disponíveis:
  - Pessoas que se engajaram com a conta (qualquer interação).
  - Pessoas que visitaram o perfil.
  - Pessoas que enviaram mensagens diretas.
  - Pessoas que salvaram publicações.
  - Pessoas que começaram a seguir a conta.

**Passo a passo para criar:**

1. Criar público personalizado → Fonte "Conta do Instagram".
2. Selecionar a conta desejada.
3. Escolher o tipo de interação.
4. Definir período de retenção.
5. Nomear e salvar o público.

---

### Público Personalizado por Página do Facebook

- Similar ao Instagram, mas para interações na página do Facebook.
- Tipos de interação:
  - Pessoas que visitaram a página.
  - Pessoas que se engajaram com publicações.
  - Pessoas que clicaram em botões de chamada para ação.
  - Pessoas que salvaram publicações.

---

### Público Personalizado por Formulário de Lead

- Segmenta pessoas que interagiram com formulários de geração de cadastro em anúncios.
- Segmentações possíveis:
  - Pessoas que abriram o formulário.
  - Pessoas que abriram mas não enviaram.
  - Pessoas que abriram e enviaram o formulário.
- Pouco utilizado no mercado de infoprodutos.

---

### Outras Fontes de Público Personalizado

- **Atividade em Apps:** usuários de aplicativos móveis (não aplicável para infoprodutos).
- **Catálogo:** interações com catálogo de produtos (mais usado em e-commerce).
- **Atividades Offline:** conversões no mundo físico, para quem faz atendimento tradicional (não muito usado em infoprodutos).
- **Eventos, Compras, Experiências Instantâneas:** raramente usados no contexto de infoprodutos.

---

### Público Semelhante (Lookalike)

- Estratégia para **expansão de alcance** baseada em características de um público base qualificado.
- Cria audiências com comportamento e dados demográficos similares ao público original.
- Fontes para criação:
  - Origem com base em valor (ex: Pixel, comportamento de compra).
  - Outras fontes (ex: lista de clientes, público personalizado).
- **Percentual de similaridade:**

| Percentual | Característica                 | Tamanho da Audiência |
|------------|-------------------------------|---------------------|
| 1%         | Máxima similaridade            | Menor               |
| 2-5%       | Alta similaridade              | Médio               |
| 6-10%      | Menor similaridade (mais amplo)| Maior               |

- Recomenda-se iniciar com percentuais menores para maior precisão.
- Não é possível criar públicos semelhantes duplicados com mesma origem, país e percentual.

**Passo a passo para criar:**

1. Criar público → Público Semelhante.
2. Selecionar fonte (pixel ou público personalizado).
3. Definir localização geográfica (ex: Brasil).
4. Configurar percentual de similaridade (ex: 0-1%).
5. Criar público.
6. Aguardar processamento.

---

### Público Salvo

- Serve para armazenar configurações de segmentação usadas com frequência.
- Permite combinar públicos personalizados, demografia, interesses e comportamentos.
- Facilita a reutilização e otimização do tempo na criação de campanhas.

**Elementos configuráveis:**

- Localização geográfica
- Faixa etária (idade mínima e máxima)
- Gênero (homens, mulheres ou ambos)
- Idioma
- Públicos a incluir (personalizados, semelhantes)
- Públicos a excluir
- Direcionamento detalhado (interesses, comportamentos)

**Passo a passo para criar:**

1. Criar público → Público Salvo.
2. Nomear o público.
3. Configurar localização, idade, gênero, idioma.
4. Adicionar públicos personalizados ou semelhantes.
5. Definir exclusões.
6. Configurar direcionamento detalhado.
7. Salvar público.

**Aplicação:**  
Ao criar conjuntos de anúncios, basta selecionar o público salvo para carregar todas as configurações pré-definidas, agilizando o processo.

## Considerações Estratégicas

- O domínio operacional da plataforma é fundamental, mas o diferencial real está na **estratégia** para encontrar o melhor público.
- É essencial combinar o conhecimento técnico com a criação de **criativos assertivos** que atraiam a audiência correta.
- Saber onde configurar públicos é importante, mas entender **como** e **por que** segmentar é o que diferencia campanhas medianas de campanhas de sucesso.
- A retenção do público, o nível de engajamento e a qualidade da base são fatores decisivos para o desempenho das campanhas.

---

## Pontos-Chave

- **Público Personalizado:** base para remarketing e nutrição, criado a partir de comportamentos reais (visitas, interações, listas).
- **Retenção:** período máximo que o público fica armazenado para remarketing (180 dias para site, 365 para vídeos).
- **Público Semelhante (Lookalike):** expande alcance com base em públicos qualificados, percentual menor = maior similaridade.
- **Público Salvo:** otimiza tempo, armazenando configurações frequentes para reutilização.
- **Fontes variadas:** site, lista de clientes, vídeos, Instagram, Facebook, formulários, entre outros.
- **Importância da nomenclatura:** usar nomes claros e padronizados facilita a gestão (ex: "VTSD - Página de Vendas - 180 dias").
- **Match de dados:** Facebook cruza dados importados com sua base para identificar usuários.
- **Engajamento em vídeo:** maior percentual assistido indica maior interesse e permite remarketing progressivo.
- **Interações em Instagram e Facebook:** permitem segmentar públicos altamente engajados.
- **Estratégia acima da operação:** conhecer a ferramenta é necessário, mas o sucesso depende da estratégia de segmentação e criativos.
