# Apostila: Integração do Pixel do Facebook com a Hotmart

---

## Resumo

A integração do pixel do Facebook com a Hotmart permite rastrear eventos essenciais para mensuração e otimização de campanhas de tráfego pago, como visitas à página de checkout e vendas realizadas. O processo envolve a configuração do pixel dentro da Hotmart, a seleção dos eventos a serem monitorados, a definição dos métodos de pagamento a considerar e a ativação da API de Conversões para garantir maior precisão no envio dos dados. Essa conexão é fundamental para construir públicos personalizados, otimizar anúncios e medir o retorno sobre investimento (ROI) com dados confiáveis.

---

## 1. Introdução à Integração com a Hotmart

- Integração entre o pixel do Facebook (Meta) e a Hotmart conecta a plataforma de anúncios às conversões reais.
- Permite acompanhar eventos desde o acesso à página de pagamento até a confirmação da compra.
- Facilita a mensuração de resultados, otimização de campanhas e criação de públicos personalizados baseados em comportamento de compra.
- Requer configuração detalhada para garantir precisão e confiabilidade dos dados.

---

## 2. Pré-requisitos para a Integração

Antes de iniciar a configuração, certifique-se de possuir:

- Conta ativa na Hotmart.
- Produto criado e configurado na Hotmart.
- Acesso ao Gerenciador de Eventos do Facebook (Meta Business Suite).
- Pixel do Facebook previamente criado.

---

## 3. Acessando a Ferramenta de Pixel de Rastreamento na Hotmart

### Passo a passo:

1. Acesse sua conta na Hotmart.
2. No menu lateral esquerdo, clique em **Ferramentas**.
3. Selecione **Ver todas** para visualizar todas as ferramentas disponíveis.
4. Utilize o campo de busca para localizar a funcionalidade **Pixel de Rastreamento**.
5. Ao acessar, selecione o produto para o qual deseja configurar o pixel (cada produto tem configuração individual).
6. A Hotmart exibirá as opções de integração disponíveis.

### Plataformas compatíveis para integração:

| Plataforma         | Finalidade                                   |
|--------------------|----------------------------------------------|
| Facebook + Instagram | Rastreamento de conversões para Meta Ads    |
| Google Ads          | Rastreamento de conversões para Google Ads  |
| Google Analytics 4  | Análise de comportamento e métricas          |
| TikTok Ads          | Rastreamento de conversões para TikTok       |
| Bing Ads            | Rastreamento para Microsoft Advertising      |
| Taboola             | Rastreamento para campanhas de native ads    |
| Outbrain            | Rastreamento para campanhas de native ads    |
| Twitter (X) Ads     | Rastreamento de conversões para Twitter (X)  |

Para este manual, o foco é a integração com **Facebook + Instagram**.

---

## 4. Obtendo o ID do Pixel no Facebook

### Procedimento para localizar o ID do pixel:

1. Acesse o **Gerenciador de Eventos** no Facebook (Meta Business Suite).
2. Na seção **Fontes de Dados**, selecione o pixel desejado.
3. O ID do pixel aparece na lateral direita como uma sequência numérica única.
4. Copie esse ID.
5. Volte para a Hotmart e cole o ID no campo solicitado.
6. Clique em **Continuar** para avançar.

---

## 5. Configuração dos Eventos de Rastreamento na Hotmart

### Eventos disponíveis para seleção:

| Evento                      | Descrição                                      | Recomendação       |
|-----------------------------|------------------------------------------------|--------------------|
| Vendas realizadas            | Dispara quando uma compra é concluída          | Obrigatório        |
| Visitas na página de pagamento (Initiate Checkout) | Dispara quando alguém acessa o checkout | Obrigatório        |
| Visitas na página do produto Hotmart | Dispara em acessos à página nativa da Hotmart | Opcional*          |

\* A página do produto Hotmart refere-se à página nativa da plataforma, não à página de vendas personalizada. Se usar página externa, deixar desmarcado.

---

## 6. Configuração do Método de Pagamento para Confirmação de Vendas

### Opções disponíveis:

| Opção                  | Métodos Incluídos                         | Comportamento                                    |
|------------------------|------------------------------------------|-------------------------------------------------|
| Todos os métodos       | Cartões, boletos, Pix, saldo Hotmart     | Contabiliza todas as transações, incluindo boletos e Pix não pagos |
| Apenas pagamentos imediatos | Cartões de crédito, débito e saldo Hotmart | Contabiliza somente transações com confirmação instantânea |

### Recomendações:

- Selecionar **apenas pagamentos imediatos** para evitar contabilização de boletos ou Pix pendentes como vendas efetivas.
- Isso mantém a atribuição e mensuração das campanhas mais precisas.

---

## 7. Diferenciação de Eventos por Tipo de Pagamento

- Possibilidade de diferenciar pagamentos imediatos e não imediatos.
- Ao ativar, a Hotmart envia evento específico chamado **Payment Generated** para pagamentos não imediatos (boletos, Pix).
- Benefícios:
  - Identificar boletos gerados vs. pagos.
  - Criar estratégias de remarketing para compradores que geraram boleto mas não pagaram.
  - Organizar e segmentar análise de dados.

---

## 8. Configuração do Valor da Transação

### Opções para envio do valor da venda:

| Opção                  | Funcionamento                                   |
|------------------------|------------------------------------------------|
| Valor da transação     | Envia o valor real pago pelo cliente            |
| Valor específico definido | Envia um valor fixo predeterminado             |

### Recomendações:

- Utilizar **valor da transação** para garantir que o Facebook receba dados precisos, mesmo com ofertas promocionais ou diferentes faixas de preço para o mesmo produto.

---

## 9. Configuração da API de Conversões

### O que é a API de Conversões?

- Envio de eventos diretamente do servidor da Hotmart para o Facebook.
- Complementa o rastreamento via navegador (web).
- Garante maior precisão e redundância no envio dos dados.

### Benefícios da API de Conversões:

- Redundância: eventos enviados via navegador e servidor.
- Maior precisão: reduz perdas causadas por bloqueadores de anúncios e configurações de privacidade.
- Envio de parâmetros adicionais: método de pagamento, valor, dados para identificação do comprador.

---

## 10. Obtenção e Configuração do Token de Acesso para API

### Passo a passo para gerar e inserir o token:

1. No Facebook, acesse o **Gerenciador de Eventos**.
2. Vá para **Fontes de Dados** e selecione o pixel desejado.
3. Clique em **Configurações** do pixel.
4. Role a página até a seção de geração de token.
5. Clique em **Gerar Token** para criar um código único.
6. Copie o token gerado.
7. Volte para a Hotmart e cole o token no campo correspondente.
8. Utilize a opção de **Verificação do Token** para confirmar a conexão (indicador verde).
9. Clique em **Salvar** para finalizar a configuração.

### Observações importantes:

- Armazene o token em local seguro, pois pode ser alterado com o tempo.
- Ter backup facilita atualizações e manutenção da integração.

## 12. Considerações Finais

- Integração entre Hotmart e Facebook Pixel é essencial para rastreamento preciso de conversões.
- Alimenta o algoritmo do Facebook com dados reais para otimização das campanhas.
- Permite construir públicos personalizados e semelhantes (Lookalike) baseados em compradores reais.
- Processo simples, porém estratégico para decisões fundamentadas em dados confiáveis.
- Garantir backup do token e revisões periódicas mantém a integração estável e eficiente.

---

## Pontos-Chave

- O ID do pixel conecta o Facebook às conversões na Hotmart.
- Selecionar eventos corretos (vendas e initiate checkout) é fundamental para mensuração.
- Preferir pagamentos imediatos evita distorções na atribuição.
- Diferenciar pagamentos não imediatos permite remarketing mais eficaz.
- API de Conversões aumenta a precisão e reduz perdas de dados.
- Token da API deve ser gerado, verificado e armazenado com segurança.
- Configuração correta impacta diretamente no ROI e na otimização das campanhas.
