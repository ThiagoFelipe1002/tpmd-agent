# Apostila: Eventos e Extensões no Facebook e Google para Tráfego Pago

---

## Resumo

Eventos são comportamentos específicos do usuário dentro do site que precisam ser monitorados para otimizar campanhas de tráfego pago. A instalação correta de pixels e tags, como o Pixel do Facebook e o Google Tag Manager, é fundamental para capturar esses eventos. Extensões do navegador Google Chrome, como o Google Tag Assistant e o Meta Pixel Helper, permitem verificar em tempo real se as tags e eventos estão sendo acionados corretamente. Monitorar eventos como visualização de página, cliques, rolagem e início de checkout possibilita decisões baseadas em dados reais, aumentando a eficiência das campanhas.

---

## 1. Conceito de Eventos

- **Eventos** são ações específicas realizadas pelo visitante dentro do site, como:
  - Visualização de página (PageView)
  - Reprodução de vídeo (Play Video)
  - Rolagem da página (Scroll Depth)
  - Clique em botão (Click)
  - Início de checkout (Initiate Checkout)
  - Compra (Purchase)
  - Geração de lead (Lead)

- O objetivo do monitoramento de eventos é obter dados concretos sobre o comportamento do usuário para otimizar campanhas de tráfego pago.

- A captura desses eventos depende da instalação correta de pixels e tags nas páginas.

---

## 2. Importância das Extensões para Verificação de Tags e Eventos

- Extensões para Google Chrome funcionam como ferramentas de inspeção que mostram quais tags e eventos estão ativos em uma página.

- Permitem verificar se os pixels e tags estão instalados e disparando corretamente, evitando erros que comprometem a performance das campanhas.

- As principais extensões recomendadas são:

  | Extensão             | Plataforma                   | Função Principal                                      |
  |----------------------|-----------------------------|------------------------------------------------------|
  | Google Tag Assistant  | Google Ads, Analytics, GTM   | Verificar todas as tags do ecossistema Google        |
  | Meta Pixel Helper     | Facebook/Meta Pixel          | Verificar instalação e disparo de eventos do Pixel   |

---

## 3. Instalação das Extensões no Google Chrome

### 3.1 Google Tag Assistant

1. Acessar o menu de extensões do Chrome (três pontinhos no canto superior direito > Extensões).
2. Abrir a Chrome Web Store.
3. Pesquisar por **Google Tag Assistant**.
4. Selecionar a extensão e clicar em **Adicionar ao Chrome**.
5. Confirmar a instalação clicando em **Adicionar extensão**.
6. Fixar a extensão na barra de ferramentas para acesso rápido (clicar no ícone de quebra-cabeça > clicar no alfinete).

### 3.2 Meta Pixel Helper

1. Acessar novamente a Chrome Web Store.
2. Pesquisar por **Facebook Pixel** ou **Meta Pixel Helper**.
3. Selecionar a extensão correta.
4. Clicar em **Adicionar ao Chrome** e confirmar.
5. Fixar a extensão na barra de ferramentas para facilitar o uso.

---

## 4. Utilização Prática das Extensões

### 4.1 Meta Pixel Helper

- Ao acessar uma página com Pixel do Facebook instalado, a extensão exibe:

  - Número total de eventos disparados.
  - Quantidade de pixels distintos presentes na página.
  - Tipos de eventos acionados (ex: PageView).
  - Parâmetros enviados junto com cada evento.

- Mesmo que os pixels estejam configurados dentro do Google Tag Manager, o Meta Pixel Helper os detecta normalmente.

- Exemplo prático:

  - Ao entrar em uma página, aparecem 5 eventos acionados.
  - É possível visualizar 4 pixels diferentes ativos.
  - O evento PageView indica uma visita à página.
  - Parâmetros adicionais podem ser visualizados para cada evento.

### 4.2 Google Tag Assistant

- Exibe todas as tags do ecossistema Google encontradas na página, incluindo:

  - Tags do Google Ads.
  - Tags do Google Analytics GA4 (versão atual).
  - Contêineres do Google Tag Manager (GTM) ativos, identificados por códigos únicos.

- Permite verificar quais pixels e tags estão sendo gerenciados dentro de cada contêiner do Tag Manager.

- Exemplo prático:

  - Visualização de tags do Google AdWords, Analytics e múltiplos GTMs.
  - Confirmação de que pixels do Facebook estão dentro do GTM.
  - Permite acompanhar se o GTM está funcionando corretamente.

---

## 5. Exemplos de Eventos em Ação

### 5.1 Evento de Reprodução de Vídeo

- Ao clicar para reproduzir um vídeo incorporado, um evento de vídeo é disparado automaticamente.
- A extensão registra o evento e acompanha métricas como percentual de vídeo assistido.
- Permite análises avançadas sobre o engajamento do usuário com conteúdo audiovisual.

### 5.2 Evento de Rolagem de Página (Scroll Depth)

- Eventos são disparados em marcos percentuais da rolagem da página:

  | Percentual de Rolagem | Evento Disparado   |
  |----------------------|-------------------|
  | 10%                  | ScrollDepth 10%   |
  | 25%                  | ScrollDepth 25%   |
  | 50%                  | ScrollDepth 50%   |
  | 75%                  | ScrollDepth 75%   |
  | 100%                 | ScrollDepth 100%  |

- Indica até onde o visitante navega no conteúdo, ajudando a identificar interesse e pontos de abandono.

### 5.3 Evento de Clique em Botão

- Quando o usuário clica em um botão de ação (ex: "Quero fazer minha primeira venda"), o evento de clique é registrado.
- Permite identificar interações específicas e medir conversões parciais.

### 5.4 Evento Initiate Checkout

- Ao preencher um formulário de pré-checkout (ex: e-mail, telefone) e avançar para a página de pagamento, o evento **Initiate Checkout** é disparado.
- Sinaliza intenção concreta de compra.
- Fundamental para estratégias de remarketing e otimização de conversões.

---

## 6. Monitoramento e Otimização com Eventos

- Acompanhar eventos permite entender o comportamento do usuário dentro do site.
- Com dados reais, é possível:

  - Identificar gargalos no funil de vendas.
  - Criar públicos personalizados baseados em ações específicas.
  - Otimizar campanhas focando em eventos de maior valor.
  - Realizar testes e ajustes fundamentados em dados concretos.

- Essa metodologia é aplicável a qualquer plataforma de tráfego pago, desde que os eventos estejam configurados e monitorados corretamente.

---

## 7. Passo a Passo para Verificar Eventos e Tags

1. Instalar as extensões Google Tag Assistant e Meta Pixel Helper no Chrome.
2. Acessar a página do site onde os pixels e tags foram instalados.
3. Observar os ícones das extensões para verificar:

   - Quais pixels e tags estão ativos.
   - Quais eventos foram disparados.
   - Parâmetros enviados em cada evento.

4. Testar interações na página para confirmar disparo de eventos, como:

   - Reproduzir vídeos.
   - Rolar a página em diferentes percentuais.
   - Clicar em botões de ação.
   - Preencher formulários e avançar no funil.

5. Confirmar que o Google Tag Manager está gerenciando corretamente as tags e pixels.

6. Corrigir possíveis falhas de instalação ou disparo com base nas informações fornecidas pelas extensões.

---

## Pontos-Chave

- Eventos são essenciais para entender o comportamento do usuário e otimizar campanhas de tráfego pago.
- Pixels e tags devem ser instalados corretamente para capturar eventos.
- Google Tag Assistant e Meta Pixel Helper são ferramentas indispensáveis para verificar a instalação e disparo de tags e eventos.
- Monitorar eventos como PageView, Play Video, Scroll Depth, Click e Initiate Checkout permite decisões baseadas em dados reais.
- O acompanhamento contínuo dos eventos possibilita identificar gargalos, criar públicos personalizados e otimizar campanhas.
- Mesmo quando pixels do Facebook estão dentro do Google Tag Manager, o Meta Pixel Helper os detecta normalmente.
- Testar eventos na prática, realizando ações no site, é fundamental para garantir o funcionamento correto do rastreamento.
